Classe

  • Pode ser usada diversas vezes
  • Ex: Passageiros do sexo masculino no trem
  • O . define q é uma classe
  • Para usar a classe poe dentro da tag class="nome_da_class"
  • Geralmente trabalha a exceção ex: tds li red e vc quer alguns blue

ID

  • Só pode ser usado uma única vez
  • Ex: Passageiro 1 do trem
  • O # define que é um id
  • Usa 1x por página, para estruturar o site, põe no body. Ex: id="principal"
  • Vc até pode usar id em mais de uma tag, mas n é a recomendação; Se precisar use classe.
  • Exceção: vc estruturar seu site e usar ids para cada estrutura (ex: cabecalho e conteudo principal)
